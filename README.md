# OFDM Simulation in MATLAB

This repository contains a full MATLAB implementation of **Orthogonal Frequency Division Multiplexing (OFDM)**, including transmitter modeling, cyclic prefix insertion, multipath channel simulation, noise addition, FFT-based demodulation, symbol detection, and bit error estimation.
